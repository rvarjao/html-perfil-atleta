# 📚 DESCRIÇÃO DA ATIVIDADE

## 🧾 Título
**Criação de Página de Perfil de Atleta com HTML e CSS**

---

## 🎯 Objetivo

Nesta atividade, você irá desenvolver uma página web simples utilizando HTML e CSS. O foco da atividade é:

- Estruturar uma página com HTML  
- Utilizar **ids** e **classes** corretamente  
- Aplicar estilos com CSS  
- Melhorar o visual de uma página  

---

## 🧠 Conteúdos trabalhados

- HTML básico  
- CSS básico  
- Seletores:
  - `#id`
  - `.class`
- Estilização de elementos  

---

## 🛠️ Instruções

1. Baixe os arquivos:
   - `index.html`
   - `style.css`

2. Abra o arquivo `index.html` no navegador.

3. Complete o HTML:
   - Coloque seu nome  
   - Escolha um esporte  
   - Escreva uma descrição  
   - Complete as informações (idade, cidade, etc.)  
   - Adicione pelo menos **3 habilidades**  

4. Edite o CSS:
   - Complete os valores faltantes (`___`)  
   - Estilize a página  
   - Melhore o visual geral  

---

## 🎨 Desafio (opcional)

- Mude as cores da página  
- Estilize o botão  
- Deixe o perfil com aparência profissional  
- Crie um novo estilo diferente  

---

## 📸 Entrega

Você deve enviar:

- Arquivo `index.html`  
- Arquivo `style.css`  
- Um print da página no navegador
